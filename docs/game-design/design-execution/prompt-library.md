## How to Use This Library

1. Choose your tool (Midjourney / DALL-E 3 / Stable Diffusion)
2. Find the category (Character / Logo / Environment / Enemy / Boss)
3. Copy the prompt block exactly — it is production-ready
4. Paste into the tool — do not paraphrase, the wording is intentional
5. If results are unsatisfactory, check the "Troubleshooting" section per tool

**Prompt IDs:** Each prompt has an ID (e.g., `MJ-CHR-001`). Reference by ID in iteration-notes.md when logging results.

```
MJ  = Midjourney
DE  = DALL-E 3
SD  = Stable Diffusion
CHR = Character
LOG = Logo
ENV = Environment
ENM = Enemy
BOS = Boss
```

**DE-CHR-001** — Main character concept art

```
Create a pixel art character concept for a humanoid combat robot with these exact specifications:

Body: Compact proportions. Slightly oversized head relative to body, similar to Mega Man NES game characters from the 1980s. Short compact legs. Medium torso.

Colors (strictly 5): Dark navy-gunmetal (#1E2A38) for main armor. Near-black (#141E2A) for shadow areas on the left side and underside. Electric cyan (#40D0F0) for the visor band across the upper face. Alert red (#F01820) for two small circular eye dots inside the visor. Slate blue-gray (#4A5E76) for the right shoulder plate specular highlight.

Features: The right arm is an arm cannon — integrated into the forearm, slightly oversized, with a barrel that extends forward. The left arm is smaller and tucked inward. The asymmetry is intentional. The visor is the single most prominent feature — a bright horizontal band of cyan light.

Pose: Neutral standing, facing the viewer at a slight 3/4 angle. Confident but not aggressive.

Style: 16-bit pixel art, NES era. Flat colors. Hard edges. No gradients. No anti-aliasing. Maximum 5 colors. Think Mega Man Classic, not modern retro.

Background: Pure black (#000000).
```

```
Create a pixel art character reference sheet on a white background with these exact requirements:

Character: A compact humanoid combat robot in Mega Man NES classic style.
Colors: Dark navy armor, electric cyan visor, two red eye dots, slate-gray shoulder highlight.
Right arm is a cannon (integrated, not held). Left arm is smaller.

Show 4 views arranged horizontally: Front view / Right side view / Back view / Left side view.

Under each view, add a small label.

Style rules: 16-bit pixel art. Maximum 5 colors. No gradients. No anti-aliasing. Hard pixel edges. Flat shading only. Professional game character sheet format.

The character should be approximately 32-40 pixels tall in the reference sheet (the sheet itself can be larger for readability).
```

## Section 4 — Style Anchor System

### Core style keywords (always include these)

The following terms consistently produce the correct visual style across all tools:

```
pixel art, 16-bit, NES era, Mega Man Classic, retro game, flat shading,
limited palette, hard pixel edges, no anti-aliasing, no gradients,
strong readable silhouette, game sprite style
```

### What these terms specifically prevent

| Term | Prevents |
|------|----------|
| `no anti-aliasing` | Blurry pixel borders that ruin sprite readability |
| `limited palette` | Image generators adding extra colors |
| `NES era` | Modern retro styles (like "synthwave" or "voxel art") |
| `hard pixel edges` | Soft/painterly shading |
| `Mega Man Classic` | Wrong proportions or wrong era |
| `flat shading` | Gradient-based shading that breaks pixel art authenticity |

### What to add for darker/more serious mood

```
post-apocalyptic, industrial, cold atmosphere, operational not ruined,
machine aesthetic, no organic elements, steel and circuitry
```

### What to add for action/energy

```
action stance, motion lines, energy glow, cannon charge, combat pose
```
